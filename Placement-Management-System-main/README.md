 Placement Management System

A full-stack MERN application built to streamline the college campus recruitment process. The platform features dedicated portals for students to apply to drives, corporate companies to screen candidates, and placement admins to coordinate schedules and view performance analytics.

 Core Tech Stack
* **Frontend:** React.js & TailwindCSS
* **Backend:** Node.js & Express.js
* **Database:** MongoDB
* **Key Integrations:** JWT Auth, Cloudinary (Resume Storage), Nodemailer (Email Alerts), and Jitsi Meet (Video Interviews)
